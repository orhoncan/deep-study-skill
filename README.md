# deep-study

A Claude Code skill that turns Claude into a personalized deep study companion. It creates a tailored study plan for any thinker, topic, or field based on your specific background, then guides your reading and serves as an interactive interlocutor.

## What it does

**This skill does not replace reading.** You read the actual texts. Claude plans, connects, and discusses.

The workflow has three phases:

### 1. Study Plan

Claude builds a personalized study plan based on your academic/professional background, your existing knowledge in adjacent fields, your goal, and your available time.

The plan includes a reading sequence, focus themes per text, **bridge questions** that connect the new material to what you already know, and explicit concept mappings between your field and the study topic.

> **Example:** An economist studying Goffman gets bridge questions like "How does Goffman's 'expressions given vs. given off' map onto the distinction between intentional signals and involuntary information leaks in Spence's signaling model?"

### 2. Reading Companion

While you read, Claude serves as a knowledgeable colleague — not a teacher. You can ask for:

- **Clarification** of unclear arguments
- **Connections** to adjacent literature or your own field
- **Formalization** discussions (has this idea been modeled?)
- **Critique** of weak arguments
- **Synthesis** across texts

### 3. Notes & Continuity

Key takeaways, bridges, and open questions are accumulated and saved as `.md` files. You can continue across multiple sessions — Claude picks up where you left off.

## Installation

Copy `skill.md` into your Claude Code skills directory:

```bash
mkdir -p ~/.claude/skills/deep-study
cp skill.md ~/.claude/skills/deep-study/skill.md
```

## Usage

```
/deep-study Goffman
/deep-study mimetic desire, 2 hours, no resources
/deep-study              # starts onboarding
```

Or just tell Claude you want to study something:

```
"I want to deeply study mechanism design"
```

### Commands

| Command | Action |
|---------|--------|
| `plan` | Show study plan |
| `where am I` | Progress summary |
| `notes` | Show/save notes |
| `connect {concept}` | Bridge to your domain |
| `save` | Save to file |
| `next` | Next reading + briefing |

## How personalization works

The skill checks for a `reader_persona.md` file (generated by the [build-persona](https://github.com/orhoncan/build-persona) skill from Readwise data). If found, it silently uses your reading history, interests, and intellectual profile to generate highly specific bridges.

Without a persona, Claude asks a few background questions and builds bridges from what you tell it. The bridges will be less specific but still personalized.

## Language

The skill auto-detects language from your first message. All output — plan, notes, commands, conversation — follows your language. Supports Turkish and English natively; works in other languages too.

## Background

This skill is based on a workflow described in a reflective essay about using Claude as a learning tool. The key insight: the most valuable feature of an LLM as a learning tool is personalization. A course has a single syllabus for everyone, but Claude creates one tailored for you.

# deep-study (Turkce)

Claude'u kisisellestirilmis bir calisma arkadasina donusturen bir Claude Code skill'i. Arka planiniza gore herhangi bir dusunur, konu veya alan icin ozel bir calisma plani olusturur, okuma surecinize rehberlik eder ve interaktif bir tartisma partneri olarak calisir.

## Ne yapar

**Bu skill okumayi degistirmez.** Asil metinleri siz okursunuz. Claude planlar, baglantilar kurar ve tartisir.

Is akisi uc asamadan olusur:

### 1. Calisma Plani

Claude, sunlara gore kisisellestirilmis bir calisma plani olusturur: arka planiniz, komsu alanlardaki bilginiz, hedefiniz ve ayirabilecginiz zaman.

Plan; okuma sirasi, metin bazinda odak temalari, yeni materyali bildiginiz alanla baglayan **kopru sorulari** ve kavram eslemeleri icerir.

> **Ornek:** Goffman calisan bir ekonomist, "Goffman'in 'expressions given vs. given off' ayrimi Spence'in sinyalizasyon modelindeki kasitli sinyaller ve istemsiz bilgi sizintilari ayrimiyla nasil ortusur?" gibi kopru sorulari alir.

### 2. Okuma Eslikcisi

Okurken Claude bilgili bir meslektas olarak calisir — hoca degil. Su turlerde destek alabilirsiniz:

- Belirsiz argumanlarin **aciklanmasi**
- Komsu literatur veya kendi alaninizla **baglanti**
- **Formalizasyon** tartismasi (bu fikir modellenmi mi?)
- Zayif argumanlarin **elestirisi**
- Metinler arasi **sentez**

### 3. Notlar ve Sureklilik

Temel cikarimlar, kopruler ve acik sorular `.md` dosyalarina kaydedilir. Birden fazla oturumda devam edebilirsiniz — Claude kaldiginiz yerden devam eder.

## Kurulum

`skill.md` dosyasini Claude Code skills dizininize kopyalayin:

```bash
mkdir -p ~/.claude/skills/deep-study
cp skill.md ~/.claude/skills/deep-study/skill.md
```

## Kullanim

```
/deep-study Goffman
/deep-study mimetik arzu, 2 saat, kaynak yok
/deep-study              # tanimayla baslar
```

Ya da Claude'a bir sey calismak istediginizi soyleyin:

```
"Bir konu calismak istiyorum: fenomenoloji"
```

### Komutlar

| Komut | Islem |
|-------|-------|
| `plan` | Calisma planini goster |
| `neredeyim` | Ilerleme ozeti |
| `notlar` | Notlari goster/kaydet |
| `bagla {kavram}` | Kavramini alanina bagla |
| `kaydet` | Dosyaya kaydet |
| `sonraki` | Siradaki okuma + briefing |

## Kisisellesirme nasil calisir

Skill, `reader_persona.md` dosyasini kontrol eder (Readwise verisinden [build-persona](https://github.com/orhoncan/build-persona) skill'i ile olusturulur). Varsa, okuma gecmisinizi ve entelektuel profilinizi kullanarak cok spesifik kopruler olusturur.

Persona yoksa Claude birkac arka plan sorusu sorar. Kopruler daha az spesifik olur ama yine kisisellestirilmistir.

## Dil

Skill, ilk mesajinizdan dili otomatik algilar. Tum ciktilar — plan, notlar, komutlar, konusma — dilinizi takip eder.

## Arka plan

Bu skill, Claude'u bir ogrenme araci olarak kullanmayi anlatan bir denemeden dogdu. Temel icgoru: bir LLM'in ogrenme araci olarak en degerli ozelligi kisisellesirmedir. Bir dersin herkese tek bir mufredati vardir, ama Claude size ozel bir mufredat olusturur.
